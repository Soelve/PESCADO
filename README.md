This repository provides source code and data files pertaining to the method we have coined PhotoElectron Spectrum Calculated from Absorption via the Densitoy Operator (PESCADAO). 
In this particular case, we have used it to study the energy- and angle distributions of a hydrogen atom exposed to a linearly polarized laser pulse within the dipole approximation.

The repository includes MATLAB source files implementing two different versions of the PESCADO method:
1) The original version in which the wave packet must be simulated until virtually all of the unbound part has been absorbed
and
2) the version in which the remaining wave function after the interaction with the laser pulse is analysed by semi-analytical means.

The data files corresponds to the results published in this article:
[https://arxiv.org/abs/2510.05776](url)

See the README files within the respective directories for more detail.
